File Storage Analyser
==========================

A command line tool to learn about your data\!
-------------------------------------------------

This tool was created to provide a simple way to gather information about your file system. 
Given a path, and a set of 'plugin' analyser classes, the tool will grab the information it needs about your files under
 that path and use the analysers to generate statistics in the form of a report, containing data \& charts.
